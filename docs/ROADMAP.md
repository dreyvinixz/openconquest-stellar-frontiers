# Roadmap

## Ordem Ideal de Implementação
O objetivo principal é chegar a um MVP 0.1 jogável antes de investir em interface avançada.

1. **Validar e organizar o scaffold** (Auditoria de arquitetura)
2. **Fechar game_engine com testes**
3. **Implementar API REST de salas** (Criação e entrada em salas)
4. **Implementar persistência simples** (Armazenamento MVP em PostgreSQL com JSON state)
5. **Implementar WebSocket** (Sincronização multiplayer e ações in-game)
6. **Implementar frontend mínimo** (React + Vite + Typescript básicos para páginas)
7. **Integrar frontend + backend**
8. **Criar primeira partida jogável** (MVP 0.1)
9. **Melhorar visual espacial** (Arte e mapas complexos)
10. **Refatorar e documentar**

## MVP 0.1
O alvo primário é permitir que duas pessoas consigam:
- Abrir no navegador.
- Criar e entrar em uma sala.
- Iniciar a partida.
- Ver o mapa galáctico básico.
- Reforçar planeta.
- Invadir planeta vizinho.
- Mover frotas.
- Passar turno.
- Vencer por objetivo secreto.
